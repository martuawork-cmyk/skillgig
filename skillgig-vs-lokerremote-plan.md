# SKILLGIG.ID vs LOKERREMOTE.COM — ANALISIS & PLAN MVP 1 BULAN

**Tanggal:** 2026-07-01  
**Dokumen:** Strategic Competitive Analysis & 30-Day MVP Plan  
**Tujuan:** Menjadi platform #1 untuk "loker remote" dan "freelance remote Indonesia"

---

## 1. EXECUTIVE SUMMARY

LokerRemote.com adalah kompetitor langsung yang sudah berjalan dengan 2.400+ lowongan, 119+ perusahaan global, dan model freemium Rp 49K/bulan. Mereka menang di **volume data** dan **SEO authority**.

SkillGig.id memiliki fondasi lebih kuat di **edukasi + ekosistem** (Learn → Build → Earn), tapi kalah di **volume lowongan** dan **keyword authority remote jobs**.

**Keputusan strategis:** Jangan mengejar LokerRemote di jalur "aggregator job board murni". Bangun **differentiated platform** yang menggabungkan:  
**Remote Jobs (kurasi + API) + Kursus + Portofolio + Community** — menjadi "LinkedIn Learning + Upwork + RemoteOK" untuk pasar Indonesia.

## 2. ANALISIS KOMPETITOR: LOKERREMOTE.COM

### 2.1. Model Bisnis

| Aspek | Detail |
|-------|--------|
| **Model** | Aggregator (scraper otomatis dari Greenhouse, Lever, Dribbble, UIUXJobsBoard) |
| **Volume** | 2.400+ lowongan aktif, 119+ perusahaan global |
| **Frekuensi Update** | Setiap hari otomatis |
| **Monetisasi** | Freemium: Gratis (50 lowongan) + Pro Rp 49.000/bulan |
| **Target** | Profesional Indonesia yang ingin kerja remote di perusahaan global |
| **Value Proposition** | "Semua remote worldwide, bisa dilamar dari Indonesia" |

### 2.2. Kekuatan (Strengths)

1. **Volume Data Masif** — 2.400+ lowongan adalah social proof kuat dan SEO gold mine
2. **Kurasi Otomatis** — Update harian tanpa effort manual, pantau 60+ perusahaan
3. **Freemium Model Jelas** — Rp 49K/bulan untuk unlimited + alerts
4. **Alert Multi-Channel** — Email harian + Telegram real-time
5. **SEO Authority** — Sudah ranking untuk kata kunci "loker remote", "remote jobs Indonesia"
6. **Trust Signal** — Menampilkan logo perusahaan besar (GitLab, Stripe, Anthropic, Figma, Airbnb)
7. **Speed to Market** — Aggregator = cepat mengumpulkan data tanpa partnership

### 2.3. Kelemahan (Weaknesses) — INI PELUANG KAMU

1. **Tidak Ada Edukasi** — Hanya job board, tidak membantu user mempersiapkan skill
2. **Tidak Ada Komunitas** — Apply sendiri, tidak ada network atau peer support
3. **Apply Ke Perusahaan Langsung** — Tidak ada platform apply internal, user pindah ke situs perusahaan
4. **Tidak Ada Portofolio** — Tidak ada tempat user showcase karya
5. **Konten Kurasi Tipis** — Deskripsi lowongan dari sumber asli, tidak ada value-add
6. **Tidak Ada Tracking Apply** — User tidak tahu status lamaran mereka
7. **No Local Jobs** — Hanya global, tidak ada lowongan remote dari perusahaan Indonesia
8. **Scraper Risk** — Rawan diblokir sumber data, TOS violation, data quality inconsistent

### 2.4. Fitur LokerRemote

- Filter: Kategori, Tipe (Full-time, Contract, Part-time, Freelance, Internship), Gaji
- Search: Text-based
- Job detail: Deskripsi, Apply langsung ke situs perusahaan
- Newsletter: Mingguan (gratis), Harian (Pro), Telegram real-time (Pro)
- Bookmark: Simpan lowongan favorit (Pro)
- Keyword Alert: Notifikasi per skill/keyword (Pro)

## 4. STRATEGI DIFFERENTIATION: JANGAN JADI LOKERREMOTE #2

**JANGAN:** "Job board remote yang lebih kecil dari LokerRemote"

### 4.2. Differentiation Framework

```
LokerRemote = Pure Job Aggregator (data only)
    ↓
SkillGig = Remote Work Ecosystem (data + education + community + tools)
```

**Pilar Differentiation:**

1. **Hybrid Jobs** — Global remote (via API) + Local freelance (kurasi manual) dalam satu platform
2. **Skill-First Discovery** — User filter berdasarkan skill yang dimiliki, bukan hanya job title
3. **Learning Path** — Tiap job punya "skill gap analysis" → rekomendasi kursus untuk apply
4. **Indonesian Context** — Semua lowongan di-translate/diadaptasi untuk konteks Indonesia (gaji, jam kerja, timezone)
5. **Community Apply** — Apply bareng-bareng, review CV peer, interview prep group

## 5. STRATEGI DATA: KURASI MANUAL + API PIHAK KETIGA

### 5.1. Kenapa Tidak Aggregator Scraper?

| Risiko | Dampak |
|--------|--------|
| TOS Violation | Greenhouse, Lever, Ashby bisa blok IP, lawsuit |
| Data Quality | Deskripsi terpotong, link apply expired, duplicate |
| Maintenance | Scraper rusak tiap kali website sumber update layout |
| SEO Duplicate | Google penalize thin content/duplicate |
| Brand Image | "Cuma copy-paste", tidak ada value add |
| Technical Debt | Butuh infra scraping, proxy rotation, anti-bot |

### 5.2. Strategi Hybrid yang Direkomendasikan

```
TIER 1: API Pihak Ketiga (Gratis) → Volume Global
        Remotive API, Jobicy API, The Muse API
        
TIER 2: Kurasi Manual (Indonesia Context) → Quality & Local
        Remote jobs dari perusahaan Indonesia yang hire global
        Freelance gigs lokal (yang sudah ada, diperluas)
        
TIER 3: Community-Submitted (Moderated) → Engagement
        User submit lowongan, admin approve
        Incentivize dengan points/credits
```

### 5.3. Sumber Data API Gratis (Prioritas)

| Sumber | Endpoint | Data | Rate Limit | Keterangan |
|--------|----------|------|------------|------------|
| **Remotive** | `remotive.com/api/remote-jobs` | Jobs global, kategori, tags | ~100 req/day | API publik, tidak perlu key |
| **Jobicy** | `jobicy.com/jobs` | Remote jobs, company info | Fair use | JSON endpoint, filter by category |
| **The Muse** | `themuse.com/api/public` | Jobs, companies, advice | API key free | Tiered access, cukup untuk MVP |
| **Adzuna** | `adzuna.com/api` | Jobs semua tipe | 1.000 req/month free | API key required, global coverage |
| **FindWork** | `findwork.dev/api` | Remote dev jobs | API key | Fokus developer/tech |

### 5.4. Proses Kurasi Manual

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│  API Pull       │     │  Filter & Clean │     │  Enrich Data    │
│  (Remotive,     │  →  │  - Remote only  │  →  │  - Translate    │
│   Jobicy, etc)  │     │  - Indonesia    │     │    title        │
│                 │     │    eligible     │     │  - Add IDR      │
│                 │     │  - Remove dupes │     │    salary est   │
│                 │     │  - Valid URL    │     │  - Tag skill    │
└─────────────────┘     └─────────────────┘     └─────────────────┘
                                                          │
                                                          ↓
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│  Publish to DB  │     │  Auto-refresh   │     │  Alert System   │
│  - Supabase     │  ←  │  - Cron: 2x/day │  ←  │  - Telegram     │
│  - Indexed      │     │  - Mark expired │     │  - Email        │
│    for search   │     │  - Archive old  │     │  - Web push     │
└─────────────────┘     └─────────────────┘     └─────────────────┘
```

### 5.5. Data Enrichment (Value-Add dari Kurasi Manual)

Setiap job yang masuk harus di-enrich sebelum publish:

1. **Translate/Adaptasi** — Judul dan deskripsi di-translate ke Bahasa Indonesia (bukan pure auto-translate, tapi adaptasi)
2. **Salary Estimation IDR** — Konversi USD/EUR ke IDR dengan range yang realistis untuk Indonesia
3. **Timezone Compatibility** — Tag: "Asia-friendly", "Flexible timezone", "EU-only", "US-only"
4. **Skill Tagging** — Auto-tag skill dari deskripsi job (React, Python, SEO, dsb.)
5. **Company Info** — Tambahkan logo, website, Glassdoor rating, company size
6. **Indonesia Context** — Tambahkan catatan: "Butuh work permit?", "Contractor only?", "Full-time employee?"
7. **Difficulty Score** — Berdasarkan skill requirement: Entry (0-2 tahun), Mid (2-5), Senior (5+)

## 7. ARSITEKTUR TEKNIS MVP

### 7.1. Database Schema (Supabase)

```sql
-- Jobs table
jobs
  id: uuid
  title: string
  title_id: string              -- translated title
  description: text
  description_id: text          -- translated description
  company_id: uuid -> companies
  category: string[]             -- ['Web Dev', 'Design']
  skills: string[]               -- ['React', 'TypeScript']
  job_type: enum                 -- full-time, contract, part-time, freelance, internship
  level: enum                    -- entry, mid, senior
  salary_min: integer           -- in IDR
  salary_max: integer
  salary_currency: string        -- IDR, USD, EUR
  location: string               -- remote, worldwide, specific region
  timezone_friendly: enum        -- asia, europe, americas, flexible
  apply_url: string
  source: string                 -- remotive, jobicy, manual, user-submitted
  source_id: string              -- original ID from API
  status: enum                   -- pending, approved, rejected, expired, archived
  published_at: timestamp
  expires_at: timestamp
  created_at: timestamp
  updated_at: timestamp
  view_count: integer
  apply_count: integer
  is_featured: boolean
  enrichment_data: jsonb          -- {translated_by: 'ai', salary_estimated: true}

-- Companies table
companies
  id: uuid
  name: string
  slug: string
  logo_url: string
  website: string
  description: text
  industry: string
  size: enum                      -- startup, mid, enterprise
  glassdoor_rating: float
  is_remote_first: boolean
  jobs_count: integer
  created_at: timestamp

-- Categories table
categories
  id: uuid
  name: string
  slug: string
  icon: string
  job_count: integer
  description: text
  seo_title: string
  seo_description: text

-- Subscribers/Alerts table
subscribers
  id: uuid
  email: string
  telegram_id: string
  skills: string[]
  categories: string[]
  job_types: string[]
  levels: string[]
  salary_min: integer
  frequency: enum                -- daily, weekly, instant
  is_active: boolean
  is_pro: boolean
  stripe_customer_id: string
  created_at: timestamp

-- Bookmarks table (for authenticated users)
bookmarks
  id: uuid
  user_id: uuid -> auth.users
  job_id: uuid -> jobs
  created_at: timestamp

-- Job submissions (user-submitted)
job_submissions
  id: uuid
  user_id: uuid
  title: string
  company_name: string
  apply_url: string
  description: text
  status: enum                   -- pending, approved, rejected
  submitted_at: timestamp
  reviewed_at: timestamp
  reviewer_notes: text
```

## 14. KESIMPULAN & NEXT STEPS

### Kunci Sukses:
1. **Jangan copy LokerRemote** — mereka menang di volume aggregator. Kamu menang di **ekosistem + konteks Indonesia + edukasi**.
2. **Volume adalah credibility** — 500+ jobs dalam 1 bulan adalah MUST. Tanpa ini, user tidak akan percaya.
3. **Kurasi manual adalah differentiator** — setiap job di-enrich dengan konteks Indonesia = value yang tidak bisa di-copy oleh scraper.
4. **SEO programmatic** — 50+ landing pages dari data = traffic organik tanpa effort content writing manual.
5. **Alert system** — email + Telegram = retention. LokerRemote punya ini, kamu harus punya juga.
6. **Harga Pro lebih murah** — Rp 39K vs Rp 49K = easy switch reason.
